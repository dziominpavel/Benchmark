Проверка устаревания задач — запуск check_stale.py, показ отчёта.

**Когда использовать:** опционально, по запросу пользователя.
Не часть основного workflow (01→05), а отдельная проверка.

## Шаги

1. **Запусти проверку:**
   ```bash
   python tools/check_stale.py
   ```

2. **Покажи отчёт:**
   - Прочитай вывод скрипта
   - Покажи таблицу устаревания задач
   - Подсвети задачи где `baseline_commit` сильно отстал от HEAD проекта

3. **Сообщи:**
   ```
   ✓ Проверка устаревания завершена
   <N> задач актуальны
   <M> задач устарели (baseline_commit старее HEAD на X коммитов)
   ```

## Правила

- **Опционально** — не обязательно для прогона
- **Устаревание ≠ ошибка** — задача может быть устаревшей но всё ещё валидной
- **Запускай между прогонами** — не во время выполнения участников
