# Benchmark Skills

Skills для автоматизации workflow бенчмарка в Windsurf/Devin Desktop IDE.

## Порядок использования

| # | Skill | Шаг | Что делает |
|---|-------|-----|-----------|
| 00 | `benchmark-00-status` | — | Показывает прогресс, подсказывает следующий шаг. **Доступен ВСЕГДА.** |
| 01 | `benchmark-01-gen-task` | 1 | Диалоговая генерация задачи (по одной) |
| 02 | `benchmark-02-resolve` | 2 | Golden answer для bugfix (промпт для resolver) |
| 03 | `benchmark-03-run` | 3 | Запуск участника (изоляция + промпт) |
| 04 | `benchmark-04-judge` | 4 | Запуск судьи (изоляция + промпт) |
| 05 | `benchmark-05-aggregate` | 5 | Агрегация → leaderboard |
| 06 | `benchmark-06-stale` | — | Опционально: проверка устаревания задач |

## Паттерн работы

```
[IDE-чат] → [Чат с моделью] → [IDE-чат] → [Чат с моделью] → ... → [IDE-чат]
```

1. В IDE-чате вызываешь skill → получаешь промпт
2. Открываешь чат с моделью → вставляешь промпт → ждёшь
3. Возвращаешься в IDE-чат → пишешь "готово"
4. Skill проверяет результат, восстанавливает изоляцию
5. Повторяешь для следующей модели

## Правила

- **Один чат с моделью за раз** — изоляция работает только для одной модели
- **Всегда возвращайся в IDE-чат** после работы с моделью
- **Никогда не открывай 2 чата с моделями одновременно**
- **Запутался → `@benchmark-00-status`** — покажет где ты и что дальше
