Запуск участника — изоляция чужих прогонов и golden-answer, подготовка промпта.

**Когда использовать:** когда нужно запустить модель на выполнение задач бенчмарка.
Пользователь вызывает один раз на каждого участника.

## Шаги

1. **Прими аргумент `<model-id>`** — slug модели (например `claude-sonnet-4.5`).

2. **Запусти изоляцию:**
   ```bash
   python tools/isolate.py hide --task all --except <model-id>
   ```
   Это скрывает:
   - Чужие прогоны (`runs/<task>/<other-model>/`)
   - Golden-answer (`tasks/bugfix/*/golden-answer/`)

3. **Собери список задач:**
   - Прочитай `tasks/` — все `task.md` файлы
   - Для каждой задачи извлеки: id, category, project, baseline_commit, title

4. **Подготовь промпт для участника:**
   ```
   Ты — участник бенчмарка. Прочитай AGENTS.md.

   Сначала зарегистрируйся:
   python tools/register_model.py --auto --id <model-id> --name "<Name>"

   Выполни задачи: <список task-id>

   Для каждой задачи:
   1. Прочитай tasks/<cat>/<task-id>/task.md
   2. cd C:/projects/<Project> && git checkout <baseline_commit>
   3. Выполни задачу
   4. Сохрани прогон:
      runs/<task-id>/<model-id>/run.md (по шаблону runs/_TEMPLATE.md)
      runs/<task-id>/<model-id>/diff.patch (git diff)
   5. Очистка: git checkout . && git clean -fd
   6. Переходи к следующей задаче

   Задачи:
   - tasks/<cat>/<task-id-1>/task.md (<title-1>)
   - tasks/<cat>/<task-id-2>/task.md (<title-2>)
   ...

   После всех задач — сообщи "готово, N прогонов созданы".
   ```

5. **Покажи промпт пользователю:**
   ```
   ПРОМПТ ДЛЯ УЧАСТНИКА (скопируй в новый чат с <model-id>):

   <промпт>

   Когда модель закончит — напиши "готово".
   ```

6. **Жди подтверждения "готово" от пользователя.**

7. **Восстанови изоляцию:**
   ```bash
   python tools/isolate.py restore
   ```

8. **Проверь результат:**
   - Для каждой задачи: `runs/<task-id>/<model-id>/run.md` существует?
   - Для feature/bugfix/refactor: `diff.patch` существует?
   - Если чего-то нет — сообщи

9. **Сообщи:**
   ```
   ✓ <model-id> выполнено (N/M прогонов)
   Дальше: @benchmark-03-run <next-model> (следующий участник)
           или @benchmark-04-judge <judge-id> (все участники готовы)
   ```

## Правила

- **Один участник за раз** — не запускай 2 модели одновременно
- **Изоляция обязательна** — без `hide` участник видит чужие прогоны
- **Restore обязателен** — после каждой модели восстанавливай
- **Проверяй файлы** — если прогонов нет, что-то пошло не так
