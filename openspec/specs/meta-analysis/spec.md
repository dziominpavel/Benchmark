## Purpose

Мета-анализ качества задач, критериев рубрикатора и стабильности моделей.
Анализ по задачам (не по волнам — волн нет). Выполняется AI-моделью по
запросу пользователя после накопления 10+ прогонов.

## Requirements

### Requirement: Мета-анализ после 2–3 задач

После 2-3 задач (или накопления 10+ прогонов) система MUST поддерживать
мета-анализ: анализ качества задач, критериев рубрикатора и стабильности
моделей. Мета-анализ выполняется AI-моделью по запросу пользователя. Анализ
по задачам (не по волнам — волн нет).

#### Scenario: Мета-анализ запущен

- **WHEN** накоплено 10+ прогонов по 2-3 задачам
- **THEN** пользователь MAY запустить мета-анализ
- **AND** результат сохраняется в `meta-analysis/<date>-summary.md`

### Requirement: Отбраковка неразличающих задач

Мета-анализ MUST выявлять задачи, не различающие модели: все получают 5/5
(слишком лёгкая) или все 0–1 (слишком сложная). Такие задачи помечаются
`non-discriminating`.

#### Scenario: Все 5/5

- **WHEN** все участники получили нормализованный балл ≥ 4.8 по задаче F-001
- **THEN** мета-анализ помечает F-001 как `non-discriminating: too-easy`

### Requirement: Анализ критериев рубрикатора

Мета-анализ MUST выявлять критерии рубрикатора, которые шумят или не различают
модели.

#### Scenario: Критерий не различает

- **WHEN** по пункту «Заметки» все участники получают 4–5 от всех судей
- **THEN** мета-анализ помечает пункт как `low-variance`

### Requirement: Анализ стабильности моделей

Мета-анализ MUST отслеживать стабильность моделей между задачами: модель,
которая на F-001 получила 4.5, а на B-001 — 2.0, помечается `unstable`.

#### Scenario: Стабильная модель

- **WHEN** модель `claude-sonnet-4.5` получила 4.3, 4.5, 4.2 на трёх задачах
- **THEN** мета-анализ помечает её как `stable`

### Requirement: Корректировка следующих задач

Результаты мета-анализа MUST использоваться для корректировки следующих задач:
заменить неразличающие задачи, уточнить рубрикатор, пересмотреть состав
участников/судей.

#### Scenario: Корректировки зафиксированы

- **WHEN** мета-анализ завершён
- **THEN** `meta-analysis/<date>-summary.md` содержит секцию «Рекомендации для следующих задач»
