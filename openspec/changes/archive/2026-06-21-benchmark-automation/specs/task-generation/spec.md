## MODIFIED Requirements

### Requirement: Диалоговая генерация задач

Генерация задач SHALL быть диалоговой (по одной задаче за раз) через skill
`benchmark-01-gen-task`, а не батч-генерацией "4 задачи за 1 сессию". Skill
ведёт диалог: пользователь описывает тему, skill предлагает формулировку,
пользователь утверждает, файл создаётся.

#### Scenario: Диалог по одной задаче

- **WHEN** пользователь говорит "сгенерируй задачу про фичу виджет для VoiceMind"
- **THEN** skill `benchmark-01-gen-task` определяет `baseline_commit` VoiceMind
- **AND** предлагает формулировку задачи (категория feature, сложность, критерии)
- **AND** спрашивает утверждение
- **WHEN** пользователь говорит "добро"
- **THEN** skill создаёт `tasks/feature/F-NNN-<slug>/task.md`
- **AND** НЕ создаёт другие задачи (по одной, не батч)

#### Scenario: Пользователь правит формулировку

- **WHEN** skill предложил формулировку
- **AND** пользователь говорит "измени сложность на 3"
- **THEN** skill обновляет формулировку
- **AND** снова спрашивает утверждение
